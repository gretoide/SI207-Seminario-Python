Enunciado de la pregunta

Implementá un programa en C que reciba como argumentos un flag y una serie de strings (la cantidad de strings no se conoce de antemano, pero al menos deberán ser 2). Lo primero que hará el programa es concatenar todos los strings en el orden que fueron pasados (utilizando un espacio para unir un string a otro) y luego, en función del flag recibido, aplicará una transformación al string resultante de la concatenación:

  • Si el flag es -i o --invert, deberá invertir el string resultante de la concatenación.
  • Si el flag es -u o --uppercase, deberá convertir a mayúsculas todos los caracteres alfabéticos del string resultante.
  • Si el flag es -l o --lowercase, deberá convertir a minúsculas todos los caracteres alfabéticos del string resultante.

Pautas

  • La implementación que realices deberá utilizar correctamente alocación dinámica de memoria.
  • El resultado deberá imprimirse a la salida estándar.
  • Los casos de error deberán manejarse como se indica en la siguiente sección.

Aclaraciones

  • El flag se recibe siempre como primer argumento del programa, el resto de los argumentos se tomarán como strings a concatenar.
  • Sólo se podrá recibir 1 (un) flag entre los argumentos.
  • Si no se recibe un flag válido, se deberá terminar la ejecución indicando el error y retornando un exit status 2.
  • Si no se reciben al menos 2 strings a concatenar, se deberá terminar la ejecución indicando el error y retornando un exit status 1.

Comentarios

Comentario:
  • El programa compila correctamente.
  • Se accede a argv[1] sin verificar que argc sea > 1.
  • Cuando se controla si se selecciona un opción válida se hace strcmp(argv[1],"-i") > 0. strcmp devuelve un valor positivo cuando el primer string es mayor. Por ende cualquier valor que se pusiera en argv que sea < no se toma como error.
  • No se hace uso de stderr para la visualización de errores. Se utiliza stdio.
  • No se reserva memoria para str. No se agrega " " entre cada argumento que se concatena.
  • No se entrega funcionalidad para resolver las opciones.
  • strupr y strlwr no son estándar.
